Configuration d'un NAS (Projet Scolaire)

Projet Réalisé
Durée 2 semaines
Contexte BTS SIO - CFA
Technologie principale Synology DSM 7
Type Individuel

Contexte du projet

Dans le cadre de mon alternance, il m'a été confié la mission de mettre en place une solution de stockage centralisée, sécurisée et capable d'héberger des services applicatifs tiers. L'objectif principal était de fournir un espace de stockage résilient pour les données du CFA, tout en permettant l'exécution de services multimédias pour des tests internes, notamment via la solution Jellyfin.

Matériel utilisé : Synology DS920+

Le choix s'est porté sur le modèle Synology DiskStation DS920+, un NAS à 4 baies adapté aux PME. Voici ses caractéristiques clés :

Timeline de réalisation

ÉTAPE 01

Installation Physique

Montage sans outil de 4 disques durs de 4 To dans les racks. Raccordement du NAS au switch réseau et à l'alimentation électrique.

ÉTAPE 02

Déploiement de DSM

Utilisation de l'outil Web Assistant (find.synology.com) pour localiser le NAS sur le réseau. Installation du système d'exploitation DiskStation Manager (DSM) 7, qui s'installe en miroir sur tous les disques pour la redondance.

ÉTAPE 03

Configuration Administrative

Définition du nom du serveur : BTS_SIO_NAS
Création d'un compte administrateur dédié : adminsio
Configuration des mises à jour de sécurité automatiques.

ÉTAPE 04

Architecture de Stockage (RAID 5 & Btrfs)

  • RAID 5 : Compromis idéal entre performance, capacité (10.9 To utiles) et sécurité (tolérance à la panne d'un disque).
  • Btrfs : Snapshots pour protection contre les ransomwares, auto-réparation des données corrompues via checksums.
ÉTAPE 05

Déploiement de Jellyfin via Docker

# Image Docker utilisée
jellyfin/jellyfin

# Volumes mappés
/config → Configuration Jellyfin
/media → Bibliothèques multimédias

# Permissions

Finalisation de la configuration via l'interface web de Jellyfin pour ajouter les bibliothèques multimédias.

Conclusion et Perspectives

Ce projet a permis de valider les compétences d'installation et d'administration d'un serveur NAS en environnement professionnel. Le Synology DS920+ est désormais opérationnel, offrant une solution de stockage résiliente et polyvalente. L'intégration de Jellyfin via Docker démontre la capacité du NAS à agir comme un serveur d'applications, dépassant son simple rôle de stockage.

Prochaines étapes :

  • Sécuriser l'accès extérieur via VPN
  • Mettre en place une stratégie de sauvegarde externalisée
  • Configurer HTTPS avec certificat SSL
